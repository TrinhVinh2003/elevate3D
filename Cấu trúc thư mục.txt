Cấu trúc thư mục:

- Folder includes:
+ connect.php: file kết nối với database
+ database.php: file chứa các hàm xử lý database
+ functions.php: file chứa các hàm chung của project
+ session.php: file chứa các hàm liên quan đến session và cookies

- Folder modules:
+ Folder auth: Folder các chức năng đăng nhập
+ Folder user: Trang của user
+ Folder admin: Trang của admin

- Folder templates
+ Folder css: chứa các file css
+ Folder fonts: chứa các font chữ
+ Folder image: chứa các ảnh
+ Folder js: chứa các file javascript
+ Folder layout: chứa các phần sử dụng chung như header và footer

- File index.php: trang chủ
- File config.php: file lưu các hằng số của project
